# Complete Application Flow Guide

## Employee Leave Application Workflow

### Step 1: Login
1. Navigate to the login page
2. Enter your username
3. Enter your Employee ID / PF Number
4. Enter your password
5. Click "Login"

**Result:** You are authenticated and redirected to the dashboard

---

### Step 2: View Dashboard & Leave Balance
1. From the navigation menu, click "Dashboard"
2. View your leave balance summary showing:
   - Total leave days available
   - Days already used
   - Remaining days
   - Breakdown by leave type

**Result:** You understand your current leave position

---

### Step 3: Apply for Leave
1. Click "Apply Leave" in the navigation menu
2. Fill in the leave application form:

#### Leave Details Section
   - **Leave Type**: Select from dropdown (Annual, Sick, Maternity, etc.)
   - **Start Date**: Click calendar to select start date
   - **End Date**: Click calendar to select end date
   - **Number of Days**: Auto-calculated based on date selection
   - **Reason** (optional): Add details about your leave

3. Review your available balance for selected leave type
4. Click "Submit Application"

**Result:** Application submitted and status shows "pending"

---

### Step 4: Upload Supporting Documents
After successful submission, you can upload supporting documents:

1. **Document Upload Section** appears with:
   - File selector button
   - File type and size restrictions information
   - Upload button

2. **Select Document**:
   - Click "Choose file..."
   - Browser file picker opens
   - Select your document

3. **Supported File Types**:
   - PDF
   - DOC, DOCX
   - JPG, JPEG, PNG
   - XLS, XLSX
   - TXT
   - Max file size: 5MB

4. **Upload Multiple Documents**:
   - Upload one document
   - Repeat process for additional supporting documents
   - Upload up to 5 documents

5. **View Uploaded Documents**:
   - Table shows all uploaded files
   - File name, size, type, uploader, and upload date
   - Download button to retrieve files
   - Delete button to remove files

**Example Documents to Upload**:
- Medical certificate (for sick leave)
- Hospital reports (for maternity leave)
- Death certificate (for bereavement leave)
- Travel bookings (for annual leave)
- Manager approval (for any leave type)

**Result:** Supporting documents are stored with application

---

### Step 5: Track Application Status

1. Click "History" to view all your leave applications
2. View status: **Pending**, **Approved**, or **Rejected**
3. See application details:
   - Leave type
   - Start and end dates
   - Days requested
   - Submission date
   - Current status

4. Filter applications by status

**Result:** You can monitor your application progress

---

## Manager/Supervisor Approval Workflow

### Step 1: View Pending Approvals
1. Click "Approvals" in the navigation menu
2. View all pending leave applications requiring your review

### Step 2: Review Application
1. Click "Review" button on pending application
2. Modal opens showing:

#### Application Details
   - Employee name and ID
   - Leave type and duration
   - Reason provided
   - **Supporting Documents** section showing all uploads
   - Download links for each document

3. Review supporting documents:
   - Download each document for inspection
   - Verify document authenticity
   - Check compliance with policies

### Step 3: Make Decision
1. Add comments about the approval decision
2. Click either:
   - **"Approve"** - Application moves to next approval level
   - **"Reject"** - Application status changes to rejected

3. Comments are recorded and can be viewed by all parties

**Result:** Application moves to next approval level or is rejected

---

## HR & Chief Officer Approval Process

### Similar to Supervisor Workflow:
1. HR reviews applications after supervisor approval
2. Chief Officer reviews after HR approval
3. Final approval updates employee's leave balance
4. Employee receives notification

**Approval Levels:**
```
Supervisor Review
    ↓ (Approve)
HR Review
    ↓ (Approve)
Chief Officer Review
    ↓ (Approve)
APPROVED ✓ (Balance Updated)

OR at any level → REJECTED ✗
```

## Important Notes

### Leave Balance Management
- Leave is deducted from balance only after final approval
- Approval at all three levels required for balance update
- If rejected, no balance deduction occurs
- Balance resets yearly (Jan 1)

### Document Guidelines
- Upload all necessary supporting documents before or after submission
- Documents help speed up approval process
- Reviewers can download documents for verification
- Keep documents under 5MB for quick upload

### Timeline
- **Pending:** Application awaiting supervisor review
- **In Progress:** Approved by supervisor/HR, awaiting next level
- **Approved:** All levels approved, leave confirmed
- **Rejected:** Not approved, resubmit if appropriate

### Approval SLA
- Supervisor: 2-3 business days
- HR: 2-3 business days
- Chief Officer: 1-2 business days
- Total typical timeframe: 5-8 business days

## Troubleshooting

### Document Upload Issues
- **"File type not allowed"**: Convert to PDF or accepted format
- **"File size exceeds limit"**: Compress image or reduce file
- **"Upload failed"**: Check internet connection, try again

### Application Issues
- **"Insufficient leave balance"**: Check available days for leave type
- **"End date before start date"**: Verify date selections
- **"Cannot submit"**: Ensure all required fields completed

### Approval Issues
- **"Cannot access approval"**: Check user role (need supervisor/HR/admin)
- **"Cannot see documents"**: Ensure you have proper permissions
- **"Approval not processing"**: Refresh page or contact admin

## Quick Tips

1. **Plan Ahead**: Submit leave applications 1-2 weeks in advance

3. **Add Detailed Reason**: Clear reason helps speed up approval
4. **Monitor Status**: Check "History" regularly for updates
5. **Keep Records**: Download approved documents for your records
6. **Contact Supervisor**: If approval takes longer than expected
